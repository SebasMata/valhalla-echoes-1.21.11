About The Mod

Welcome to Valhalla Echoes. This mod brings ancient Norse mythology right into your Minecraft world. Explore new structures, fight alongside mystical Vikings, and craft legendary equipment.

Key Features

  • Viking Villages: Discover dynamically generated settlements including big huts and social centers.
  • Mystical Equipment: Craft powerful mystical swords, axes, and full armor sets to defend yourself.
  • Unique Items: Find rare materials like Obsidian Rods, Runic Gems, and Suspicious Substances.
  • Custom Entities: Encounter Mystical Vikings and interact with them in their natural habitat.
  • Combat Enhancements: Block attacks effectively using the new Targe Shield and Horned Helmet.

Installation

Follow these simple steps to install the mod and start playing.

  • Download the Fabric Installer and install it for version 1.21.11
  • Download the Fabric API and place it in your mods folder
  • Download the Valhalla Echoes jar file
  • Place the downloaded jar file into your Minecraft mods folder
  • Launch the game and enjoy
